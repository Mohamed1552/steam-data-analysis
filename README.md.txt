# Steam Data Analysis Project

This project analyzes Steam video game data (from Kaggle dataset).  
It includes:
- Data cleaning and preprocessing.
- Exploratory data analysis (EDA) with charts.
- Insights about top developers, genres, and owners.
- A Streamlit dashboard for interactive visualization.

## How to Run
1. Clone the repo.
2. Install dependencies:
   ```bash
   pip install pandas matplotlib seaborn streamlit
